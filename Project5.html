Project overview¶

The Titanic Data Analysis project is an immersive and enlightening endeavor that delves into the intricate details of the historic RMS Titanic disaster through the lens of data analysis. This project takes on the challenge of unraveling the events surrounding one of the most renowned maritime tragedies of the 20th century, leveraging comprehensive passenger data to glean insights into the factors that determined survival rates. By employing a multi-faceted approach encompassing data exploration, visualization techniques, and potentially predictive modeling, this project aims to uncover hidden patterns, offer poignant conclusions, and contribute to a profound understanding of the impact the disaster had on passengers.

The objectives of this project includes: historical contextualization of the incident, understanding survival patterns, data visualization to convey intricate narratives and predictive modeling to forecast survival probabilities based on passenger attributes.

In order to extract meaningful insights, this project involves a systematic approach such as: data cleaning, exploratory data analysis to unveil survival patterns, feature engineering to create meaningful features from the data to enhance predictive potential, and survival analysis to examine survival rates across passenger groups.

Data Cleaning¶

We want to fill in missing age data instead of just dropping the missing age data rows. One way to do this is by filling in the mean age of all the passengers (imputation). However we can be smarter about this and check the average age by passenger class. For example:

In [28]:
plt.figure(figsize=(12, 7))
sns.boxplot(x='Pclass',y='Age',data=train,palette='winter')
Out[28]:
<AxesSubplot:xlabel='Pclass', ylabel='Age'>

We can see the wealthier passengers in the higher classes tend to be older, which makes sense. We'll use these average age values to impute based on Pclass for Age. Let's create a function to input the average values in the null values.

In [29]:
def impute_age(cols):
    Age = cols[0]
    Pclass = cols[1]
    
    if pd.isnull(Age):

        if Pclass == 1:
            return 37

        elif Pclass == 2:
            return 29

        else:
            return 24

    else:
        return Age

Apply the function

In [30]:
train['Age'] = train[['Age','Pclass']].apply(impute_age,axis=1)
